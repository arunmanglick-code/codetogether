Mongo Installation:
----------------------
https://docs.mongodb.com/manual/tutorial/install-mongodb-on-windows/

1. Install Mongo DB (mongodb.com)
2. Open Cmd & Go to c:\Program Files\MongoDB\Server\Version\bin
3. Type: mongod --dbpath "Data Folder Path"  (C:\Program Files\MongoDB\Server\4.4\data)
   (This will run MongoDB service - Keep this window as-is) 

4. Install monk - npm install monk --save
5. Open Cmd (maybe same cmd window) with path to C:\Program Files\MongoDB\Server\4.4\bin
    - Type mongo (This will run MongoDB Client - Keep this window as-is) 
    - OR Double-click on mongo.exe.
